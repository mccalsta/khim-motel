Setup Guide

How to connect real email and WhatsApp notifications

📧 Step 1: Connect Email (Free via EmailJS)

1

Create a free account at EmailJS

Go to emailjs.com and sign up for free. You get 200 emails/month free.

2

Add an Email Service

In EmailJS dashboard → Email Services → Add New Service → connect your Gmail or any email.

3

Create an Email Template

Go to Email Templates → Create New. Use these variables in your template:

Subject: New Booking from {{guest_name}} — KHIM Motel

Body:
Guest: {{guest_name}}
Phone: {{guest_phone}}
Room: {{room_type}}
Check-in: {{checkin}}
Check-out: {{checkout}}
Nights: {{nights}}
Total: UGX {{total}}
Special Requests: {{special_requests}}
4

Copy your credentials to Settings

Go to Account → API Keys and copy your Public Key. Then copy your Service ID and Template ID into the Settings page of this dashboard.

💬 Step 2: WhatsApp Notifications (No API needed)

1

Add your WhatsApp number in Settings

Enter your number with country code, e.g. +256700000000.

2

How it works

When a booking is submitted, a WhatsApp link is automatically generated. Tap it from any device and the booking details will be pre-filled in a WhatsApp message to you. No monthly fees, no API keys.

🌐 Step 3: Host Your Website

1

Upload both files to any web host

Upload index.html (main website) and admin.html (this dashboard) to your hosting provider. Recommended free option: Netlify.com — drag and drop your files.

2

Access your admin dashboard

Visit yoursite.com/admin.html to manage bookings. Default login: admin / khim2024. Change the password in the source code before going live.
